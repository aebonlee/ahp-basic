### 1. SPSS 대체 통계분석 기능 신규 구현

**목적**: 연구자가 설문조사 데이터(survey_responses)에 대해 T검정, ANOVA, 카이제곱 등 SPSS에서 수행하는 통계분석을 AHP Basic 내에서 직접 수행할 수 있도록 한다.

**구현 분석 9종**:

| # | 분석명 | 주요 출력 |
|---|--------|-----------|
| 1 | 기술통계 | 평균, 중앙값, 최빈값, 표준편차, 왜도, 첨도, 히스토그램 |
| 2 | 독립표본 T검정 | t값, df, p값, Cohen's d, 그룹별 기술통계 |
| 3 | 대응표본 T검정 | t값, df, p값, 차이 기술통계 |
| 4 | 일원분산분석(ANOVA) | F값, df, p값, η², 그룹별 기술통계 |
| 5 | 카이제곱 검정 | χ², df, p값, Cramér's V, 교차표 |
| 6 | 상관분석 | Pearson r 행렬, p값 행렬, 산점도 |
| 7 | 단순선형회귀 | R², β, t, p, 잔차도, 회귀선 |
| 8 | 크론바흐 알파 | α값, 항목 삭제 시 α, 항목-총점 상관 |
| 9 | 교차분석 | 빈도표, 비율표, 기대빈도, 잔차 |

### 4. 기술 구현 핵심

**수학 구현 (외부 라이브러리 없이 순수 JS)**:
- `gammln()` — Lanczos 근사법으로 ln(Γ(x)) 계산
- `regularizedBeta()` — 정규화 불완전 베타 함수 (Lentz 연분수법, 대칭 변환)
- `regularizedGamma()` — 정규화 불완전 감마 함수 (급수전개 + 연분수법)
- `tCDF()`, `fCDF()`, `chiSquaredCDF()` — 분포별 p값 계산

**기존 라이브러리 재사용**: Recharts (차트), xlsx (내보내기), file-saver (다운로드)

**UI 워크플로**:
```
[카드 선택] → [변수 설정] → [분석 실행] → [결과 표시]
                                          ├─ 통계표
                                          ├─ 차트 (Recharts)
                                          └─ Excel 내보내기
```
