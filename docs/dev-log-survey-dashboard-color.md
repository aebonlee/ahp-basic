# 개발일지: 설문 집계 대시보드 Q문항 컬러 변경

**날짜:** 2026-03-18
**작업자:** Claude Opus 4.6

## 개요

설문 집계 페이지의 대시보드에서 Q1부터 시작하는 질문별 응답률 카드의 컬러가 너무 다채로워
통일감이 떨어진다는 피드백을 반영하여, 다크 블루(`#1e3a5f`)로 통일했다.

## 변경 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/pages/SurveyResultPage.jsx` | 질문별 응답률 바 fill 색상 변경 |
| `src/pages/SurveyResultPage.module.css` | 질문 카드 좌측 보더 + 퍼센트 텍스트 색상 변경 |

## 상세 변경 사항

### 색상 변경 (공통: `#8b5cf6` → `#1e3a5f`)

1. **`.dashCardSm` 좌측 보더**: `border-left: 3px solid #8b5cf6` → `#1e3a5f`
2. **`.dashSmPct` 퍼센트 텍스트**: `color: #8b5cf6` → `#1e3a5f`
3. **응답률 바 fill (인라인)**: `background: '#8b5cf6'` → `'#1e3a5f'`

## 변경 전후

| 요소 | Before | After |
|------|--------|-------|
| 카드 좌측 보더 | 보라색 (#8b5cf6) | 다크 블루 (#1e3a5f) |
| 퍼센트 텍스트 | 보라색 (#8b5cf6) | 다크 블루 (#1e3a5f) |
| 응답률 바 | 보라색 (#8b5cf6) | 다크 블루 (#1e3a5f) |
