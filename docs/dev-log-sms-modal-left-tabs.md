# 개발일지: SMS 모달 좌측 패널 탭 UI 리팩토링

**날짜:** 2026-03-17
**작업자:** Claude Opus 4.6

## 개요

SMS 발송 모달의 좌측 패널을 탭 기반 UI로 리팩토링했다.
기존의 접이식(collapsible) 번호 직접 입력을 제거하고,
"평가자 목록" / "번호 직접 입력" 두 개의 탭으로 분리하여
각 기능을 더 넓은 공간에서 사용할 수 있도록 개선했다.

## 변경 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/components/admin/SmsModal.jsx` | 좌측 패널 탭 UI (`leftTab` state) + 직접 입력 탭 UI 강화 |
| `src/components/admin/SmsModal.module.css` | 패널 너비 390px, 탭 스타일, 직접 입력 UX 개선 |

## 상세 변경 사항

### 1. 좌측 패널 너비 확장 (340px → 390px)
- 필터 칩이 3줄에서 2줄로 줄어듦

### 2. 탭 기반 UI 전환
- **평가자 목록 탭**: 기존 필터 칩 + 수신자 체크리스트 + 페이지네이션
- **번호 직접 입력 탭**: 전화번호 textarea + 유효 번호 카운트 + 번호 칩 미리보기

### 3. 직접 입력 탭 UX 강화
- monospace 폰트로 번호 가독성 향상
- 좌측 초록색 보더로 입력 영역 강조
- 유효 번호를 칩 형태로 미리보기 표시
- 상태 바에 유효 번호 건수 표시

### 4. 스크롤 제거
- 좌측 패널을 flex column + overflow hidden으로 구성
- 탭 콘텐츠 영역만 내부 스크롤 처리

## UI 구조

```
좌측 패널 (390px)
├── 수신자 라벨 + N명 선택 배지
├── [평가자 목록] [번호 직접 입력]  ← 탭 바
└── 탭 콘텐츠 (overflow-y: auto)
    ├── (평가자 목록) 필터 칩 → 수신자 리스트 → 페이지네이션
    └── (번호 직접 입력) 안내문 → textarea → 유효번호 카운트 → 번호 칩 미리보기
```
