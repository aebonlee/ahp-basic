# 개발일지: SMS 직접 발송 기능 추가

**날짜:** 2026-03-17
**작업자:** Claude Opus 4.6

## 개요

SuperAdmin > SMS 관리 탭에 전화번호를 직접 입력하여 SMS를 발송할 수 있는 기능을 추가했다.
기존에는 강의 신청자 목록이나 프로젝트 평가자 목록에서만 SMS를 보낼 수 있었으나,
이제 임의의 전화번호로도 메시지를 보낼 수 있다.

## 변경 파일

| 파일 | 변경 내용 |
|------|-----------|
| `src/pages/SuperAdminPage.jsx` | `SmsTab` 컴포넌트에 직접 발송 패널 추가 |

## 상세 변경 사항

### SmsTab 직접 발송 패널

- **전화번호 입력**: textarea로 여러 번호 입력 지원 (쉼표 또는 줄바꿈 구분)
- **번호 파싱**: 하이픈(`-`)과 공백 자동 제거 후 `0XX-XXXX-XXXX` 형식 유효성 검증 (`/^0\d{9,10}$/`)
- **유효 번호 카운트**: 실시간으로 유효한 전화번호 개수 표시
- **메시지 입력**: 기존 SMS 패널과 동일한 textarea + 바이트 카운터
- **SMS/LMS 자동 구분**: EUC-KR 기준 90바이트 이하 SMS, 2000바이트 이하 LMS
- **순차 발송**: 각 번호에 대해 `sendSms()` 호출, 진행률 표시
- **결과 표시**: 발송 완료 후 번호별 성공/실패 목록 + toast 알림

### UI 구조

```
SMS 관리 탭
├── 직접 발송 패널 (신규)
│   ├── 전화번호 입력 textarea
│   ├── 유효 번호 카운트
│   ├── 메시지 입력 textarea
│   ├── 바이트 카운터 (SMS/LMS/OVER)
│   └── 발송 버튼
└── SMS 통계 테이블 (기존)
    ├── 총 발송 / 성공 / 실패 요약
    └── 발신자별 상세 테이블
```

## 기술 노트

- `sendSms({ receiver, message })` 함수를 직접 사용 (Supabase Edge Function 호출)
- 프로젝트와 무관한 직접 발송이므로 `sms_logs` DB 기록 및 SMS 할당량 차감은 하지 않음
- 기존 `smsPanel` CSS 클래스를 재활용하여 일관된 UI 유지
