### 설문 집계 페이지 — 평가완료/설문미완료 불일치 버그 수정

**문제**: 평가자가 "평가 완료" 상태인데 설문 집계(step 8) 페이지에서는 미완료로 표시되는 버그.

**원인 분석**:
- 평가자 관리: `evaluators.completed` 컬럼 기준 (SignaturePanel에서 true 설정)
- 설문 집계: `survey_responses` 테이블에서 unique `evaluator_id` 카운트 기준
- 설문 질문이 없거나, 설문 없이 평가를 완료한 경우 `survey_responses`에 레코드가 없어 카운트에서 누락됨

**수정 내용**:

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 요약 영역 | `응답 완료 / 전체 배정` 단일 카운트 | `설문 응답` + `평가 완료` 이중 카운트 표시 |
| 평가자 현황 | 없음 | 이름, 설문 응답 여부, 평가 완료 여부를 보여주는 테이블 추가 |
| 설문 질문 없을 때 | "설문 질문이 없습니다" 메시지만 표시 | 평가자 상태 테이블은 계속 표시 |

**평가자별 현황 테이블 예시**:

```
이름     | 설문 응답 | 평가 완료
홍길동   |  완료     |  완료
김철수   |  완료     |  미완료
이영희   |  미응답   |  완료     ← 이 케이스가 기존에 누락
박민수   |  완료     |  완료
최지우   |  미응답   |  미완료
```
