# 개발일지 — 세션 27 (2026-03-01)

## 메인 히어로 영역 리디자인: "Analytical Depth"

### 변경 사유
기존 히어로가 흰 배경(#ffffff)에 거의 안 보이는 glow(3~5% 불투명도)와 격자선(2.5%)으로 구성되어 밋밋했음.
실제 플랫폼은 통계 10종, 설문 빌더, 평가자 관리, 민감도 분석 등 고급 기능을 갖춘 전문 도구인데 히어로가 이를 반영하지 못함.

### 디자인 컨셉
**다크 네이비 배경 + 4색 브랜드 컬러 플로팅 오브 + AHP 계층 노드 장식 + 그래디언트 시머 타이틀 → 프리미엄 분석 플랫폼 느낌**

### 주요 변경 사항

#### 배경
- `#ffffff` → `#0a0f1e` (다크 네이비)
- 4색 radial gradient 배경 레이어 추가

#### 플로팅 오브 4개
- Blue 300px / Violet 250px / Teal 200px / Emerald 180px
- blur 60px, 독립적 drift 애니메이션 (20~28초)

#### AHP 계층 노드 장식
- 5개 원형 노드 (루트 1개 + 중간 2개 + 리프 2개)
- 4색 배분, `::after` 하향 연결선, `hierPulse` 6초 호흡 애니메이션
- 전체 컨테이너 opacity: 0.5

#### 타이틀 "AHP Basic." 그래디언트
- 기존 2색 → 4색 (Blue → Violet → Teal → Emerald)
- `gradientShift` 8초 시머 애니메이션

#### 텍스트·뱃지·CTA
- 텍스트 색상 다크 배경용으로 반전
- 뱃지: 글래스모피즘 (backdrop-filter: blur) + 초록 펄싱 도트
- CTA Primary: gradient 배경 + blue glow 그림자
- CTA Secondary: 글래스모피즘

#### 격자 패턴
- 80px 간격 (기존 120px), 4% 불투명도 (기존 2.5%)
- mask-image로 중심부만 보이고 가장자리 페이드아웃

#### CSS 애니메이션

| 키프레임 | 시간 | 용도 |
|----------|------|------|
| `orbDrift1~4` | 20~28s | 오브 drift |
| `hierPulse` / `hierPulseRoot` | 6s | 노드 호흡 |
| `gradientShift` | 8s | 타이틀 시머 |
| `badgePulse` | 2s | 뱃지 도트 |
| `scrollBounce` | 1.8s | 스크롤 (기존) |

#### 반응형 (768px 이하)
- 오브 크기 축소 + opacity 0.7
- 계층 노드 숨김
- 기존 반응형 규칙 유지
